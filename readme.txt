== Description ==

Allows you to identify structured data in Beaver Builder rows.

Requires [Beaver Builder](http://wpbeaverbuilder.com/).

= Supported Types: =
* WPHeader
* WPFooter

== Installation ==

1. Upload the ft-bb-schema folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
